# 🤖 Agentic RAG System

An advanced **Agentic Retrieval-Augmented Generation (RAG)** system that goes beyond traditional RAG by incorporating autonomous agents for intelligent reasoning, tool-calling, and verified document retrieval.

## 🚀 Features
* **Autonomous Reasoning:** Uses an agentic loop to analyze and refine user queries.
* **Multi-Step Tasking:** Can perform tool-calling to fetch, filter, and summarize data.
* **Interactive UI:** Built with **Streamlit** for a seamless user experience.
* **Optimized Performance:** Developed using **Python 3.10**.

## 🛠️ Tech Stack
* **Language:** Python 3.10
* **Models:** Gemini / Groq / Ollama
* **Frontend:** Streamlit
* **Framework:** LangChain / LlamaIndex
